# Ignite

Ignite is a daily planning app that helps you build a consistent routine through a morning check-in, task tracking, and an evening reflection. A streak system encourages you to show up every day.

---

## How it works

**Morning**
When you open Ignite, Kindly greets you by name and asks what you plan to do today. You enter your tasks as a simple list.

**During the day**
Check off tasks as you complete them. You can add or remove tasks at any time.

**Evening**
At the end of the day, Ignite invites you to review your list, check off any remaining completed tasks, and write a short note about how your day went.

**Streak**
Each day you complete both your morning check-in and your evening reflection, your streak grows by one. Missing a day resets it to zero.

---

## Getting started

**Requirements**
- Node.js 18 or higher
- npm

**Installation**

```bash
git clone https://github.com/your-username/ignite.git
cd ignite
npm install
npm run dev
```

The app runs at `http://localhost:5173`.

---

## Data & privacy

All your data (tasks, reflections, streak) is stored locally on your device. Nothing is sent to any server.
